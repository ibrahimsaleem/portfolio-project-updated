- React-Bootstrap, CSS3
- Axios, React Icons, tsparticles, typewriter-effect

### Features
- Multi-page layout (Home, About, Projects, Blog, Resume)
- Responsive UI with React-Bootstrap
- Easy content updates via `src/components/*`

### Getting Started
Prerequisites: Node.js (LTS) and Git

Install dependencies and start the dev server:

```bash
npm install
npm start
```

Open http://localhost:3000 to view it in the browser. The page reloads on edits.

### Build
```bash
npm run build
```

### Project Structure
- `public/`: static assets and HTML template
- `src/components/`: page and UI components
- `src/Assets/`: images and media
